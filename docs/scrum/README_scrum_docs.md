# Scrum & Agile Documentation

Agile/Scrum deliverables for the *Realizacja Projektu Informatycznego* (Group Project)
course, documenting how the team organised and ran the **Glaucoma Detection** project.

> **All documents in this directory are written in Polish** (course requirement).
> The English descriptions below are provided for navigation only.

## Files

| File | Document (PL) | Description |
|------|---------------|-------------|
| `project_organization_and_infrastructure.pdf` | Organizacja i infrastruktura projektu | Project setup: team members and roles, repository, tooling and infrastructure. |
| `methodology_selection_and_adaptation.pdf` | Dobór i adaptacja metodyki | Selection and adaptation of the development methodology (Scrum + Kanban) for the team. |
| `scrum_product_backlog.pdf` | Scrum: Backlog produktu | The product backlog - full list of planned features and requirements. |
| `scrum_sprint_backlog.pdf` | Scrum: Backlog sprintu | Sprint backlog (sprint 5) - items committed for the sprint. |
| `kanban_board.pdf` | Kanban: Tablica Kanban | Snapshot of the Kanban board (sprint 6) - task flow across columns. |
| `scrum_sprint_retrospective.pdf` | Scrum: Retrospektywa sprintu | Sprint retrospective (sprint 6) - what went well, what to improve. |
